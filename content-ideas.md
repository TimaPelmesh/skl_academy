## 🎮 Топ 10 тренажеров и интерактивных песочниц

### 1. Алгоритм Minimax (ИИ для игр)
**Описание:** Визуализация алгоритма minimax для игры в крестики-нолики или шахматы. Показывает дерево решений, оценку позиций, выбор оптимального хода.

**Что показывать:**
- Дерево всех возможных ходов
- Оценка каждой позиции (min/max)
- Выбор оптимального хода
- Анимация обхода дерева
- Возможность играть против ИИ

**Технологии:** JavaScript, Canvas/SVG для визуализации

---

### 2. Визуализация алгоритмов сортировки
**Описание:** Интерактивная демонстрация различных алгоритмов сортировки (пузырек, быстрая, слиянием, кучей) с визуализацией процесса.

**Что показывать:**
- Массив в виде столбцов разной высоты
- Анимация сравнений и перестановок
- Подсчет операций и времени
- Сравнение производительности алгоритмов
- Возможность менять размер массива и скорость

**Технологии:** JavaScript, Canvas, D3.js

---

### 3. Деревья и графы: интерактивный конструктор
**Описание:** Инструмент для создания и визуализации деревьев и графов с различными алгоритмами обхода.

**Что показывать:**
- Создание узлов и связей мышью
- Визуализация обхода в глубину (DFS)
- Визуализация обхода в ширину (BFS)
- Поиск кратчайшего пути (Дейкстра)
- Минимальное остовное дерево (Крускал, Прим)
- Анимация работы алгоритмов

**Технологии:** JavaScript, vis.js или D3.js

---

### 4. Симулятор сетевых протоколов
**Описание:** Визуализация работы TCP/IP, HTTP, DNS и других протоколов в реальном времени.

**Что показывать:**
- Отправка пакетов между узлами
- Разбор пакетов (заголовки, данные)
- TCP handshake (3-way handshake)
- HTTP запрос/ответ
- DNS lookup процесс
- Потеря пакетов и retransmission

**Технологии:** JavaScript, WebGL или Canvas

---

### 5. Git визуализатор
**Описание:** Интерактивная визуализация работы Git: коммиты, ветки, слияния, rebase.

**Что показывать:**
- Граф коммитов
- Создание и переключение веток
- Слияние веток (merge)
- Rebase операция
- Разрешение конфликтов
- История изменений файлов

**Технологии:** JavaScript, Git graph библиотеки

---

### 6. SQL Playground
**Описание:** Интерактивная песочница для изучения SQL с визуализацией запросов.

**Что показывать:**
- Редактор SQL запросов
- Визуализация JOIN операций
- План выполнения запроса (EXPLAIN)
- Результаты в табличном виде
- Схема базы данных
- Примеры готовых запросов

**Технологии:** JavaScript, SQL.js или подключение к БД

---

### 7. Регулярные выражения: визуализатор
**Описание:** Интерактивный инструмент для изучения и тестирования регулярных выражений.

**Что показывать:**
- Визуализация паттерна (дерево)
- Подсветка совпадений в тексте
- Группы захвата
- Пошаговое выполнение
- Примеры готовых regex
- Тестирование на разных строках

**Технологии:** JavaScript, regex-vis или собственная визуализация

---

### 8. Криптография: шифрование и хеширование
**Описание:** Демонстрация работы алгоритмов шифрования и хеширования.

**Что показывать:**
- Шифрование/дешифрование (AES, RSA)
- Хеширование (MD5, SHA-256)
- Цифровые подписи
- Симметричное vs асимметричное шифрование
- Визуализация процесса
- Примеры взлома слабых алгоритмов

**Технологии:** JavaScript, Web Crypto API

---

### 9. Парсер и AST визуализатор
**Описание:** Визуализация работы парсера и построения абстрактного синтаксического дерева.

**Что показывать:**
- Ввод кода (Python/JavaScript)
- Лексический анализ (токены)
- Синтаксический анализ
- Построение AST
- Визуализация дерева
- Выполнение кода

**Технологии:** JavaScript, Babel parser или acorn

---

### 10. Симулятор виртуальной памяти и кэша
**Описание:** Визуализация работы оперативной памяти, кэша процессора, алгоритмов замещения страниц.

**Что показывать:**
- Структура памяти (стек, куча)
- Работа кэша (L1, L2, L3)
- Алгоритмы замещения (LRU, FIFO)
- Фрагментация памяти
- Управление памятью в разных языках
- Утечки памяти

**Технологии:** JavaScript, Canvas для визуализации
