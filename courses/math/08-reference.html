Все основные формулы

Арифметика. Дробь a/b + c/d = (ad+bc)/(bd). a/b · c/d = (ac)/(bd). a/b : c/d = (ad)/(bc). Процент: p% от N = N·p/100. Число по его части: если a = b·x, то b = a/x. Степень: aⁿ·aᵐ = aⁿ⁺ᵐ, aⁿ/aᵐ = aⁿ⁻ᵐ, (aⁿ)ᵐ = aⁿᵐ, (ab)ⁿ = aⁿbⁿ. Корень: (√a)² = a при a ≥ 0; √(a²) = |a|; ⁿ√(ab) = ⁿ√a·ⁿ√b.

Алгебра: ФСУ и квадратное уравнение

(a±b)² = a²±2ab+b²; a²−b² = (a−b)(a+b); (a±b)³ = a³±3a²b+3ab²±b³; a³±b³ = (a±b)(a²∓ab+b²).

ax²+bx+c = 0: D = b²−4ac; x₁,₂ = (−b±√D)/(2a); x₁+x₂ = −b/a, x₁x₂ = c/a.

Функции. Линейная: y = kx+b, k = tg α. Парабола y = ax²+bx+c: вершина x₀ = −b/(2a), y₀ = −D/(4a). Расстояние между точками A(x₁;y₁) и B(x₂;y₂): AB = √((x₂−x₁)²+(y₂−y₁)²).

Геометрия. Треугольник: S = ½ah = ½ab·sin γ = √(p(p−a)(p−b)(p−c)), p = (a+b+c)/2. Прямоугольный: c² = a²+b², S = ½ab. Параллелограмм: S = ah = ab·sin α. Трапеция: S = (a+b)h/2, средняя линия m = (a+b)/2. Круг: L = 2πR, S = πR². Окружность: вписанный угол равен половине дуги.

Стереометрия. Призма V = Sосн·h. Пирамида V = (1/3)Sосн·h. Цилиндр V = πR²h, Sбок = 2πRh. Конус V = (1/3)πR²h, Sбок = πRl. Шар V = (4/3)πR³, S = 4πR².

Тригонометрия. sin²α+cos²α = 1; tg α = sin α/cos α. sin(π/2−α) = cos α, cos(π/2−α) = sin α.

Прогрессии. АП: aₙ = a₁+(n−1)d, Sₙ = (a₁+aₙ)n/2. ГП: bₙ = b₁qⁿ⁻¹, Sₙ = b₁(qⁿ−1)/(q−1), при |q|<1 S∞ = b₁/(1−q).

Производная (ЕГЭ). (xⁿ)' = nxⁿ⁻¹, (eˣ)' = eˣ, (ln x)' = 1/x, (sin x)' = cos x, (cos x)' = −sin x. Касательная: y = f(x₀)+f'(x₀)(x−x₀).
