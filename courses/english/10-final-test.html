Тест 2: Деловая переписка

Задание

Напишите email менеджеру с просьбой о переносе дедлайна проекта на неделю. Используйте:

  • Правильное приветствие
  • Вежливую просьбу
  • Объяснение причины
  • Профессиональную подпись

Пример ответа

Показать пример 
Subject: Request for Deadline Extension - Project Alpha

Dear Mr. Johnson,

I hope this email finds you well.

I am writing to request an extension for the Project 
Alpha deadline. Due to unexpected technical issues 
with the third-party API integration, we need 
additional time to ensure the quality of the deliverable.

Would it be possible to extend the deadline by one 
week, from February 1 to February 8?

I apologize for any inconvenience this may cause and 
appreciate your understanding.

Kind regards,
Anna Smith
Software Developer

Тест 4: Документация

Прочитайте и ответьте

POST /api/v2/users

Description: Creates a new user account.

Request Body (JSON):
{
  "email": "user@example.com",
  "password": "securePassword123",
  "name": "John Doe"
}

Response:
  - 201 Created: User successfully created
  - 400 Bad Request: Invalid input data
  - 409 Conflict: Email already exists

Note: Password must be at least 8 characters.
This endpoint is rate-limited to 10 requests per minute.

Вопросы:

  1. Какой HTTP метод используется?
  2. Какие поля обязательны в запросе?
  3. Что означает статус 409?
  4. Какое ограничение на пароль?
  5. Сколько запросов можно сделать в минуту?

Ответы

Показать ответы
  1. POST
  2. email, password, name
  3. Email уже существует (Conflict)
  4. Минимум 8 символов
  5. 10 запросов в минуту
