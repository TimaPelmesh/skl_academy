Добро пожаловать на курс Python!

Этот курс предназначен для начинающих программистов, которые хотят освоить один из самых популярных языков программирования.

Что вы узнаете:

  • Основы синтаксиса Python
  • Работу со структурами данных
  • Принципы ООП
  • Работу с файлами и внешними модулями
  • Продвинутые концепции языка

Структура курса:

Курс разделен на 8 модулей, каждый из которых содержит практические задания и примеры кода. Рекомендуется проходить модули последовательно.

Требования:

  • Базовые знания компьютерной грамотности
  • Установленный Python 3.10 или новее
  • Текстовый редактор (VS Code, PyCharm или Sublime Text)

Для начала обучения выберите тему в боковом меню слева. Удачи в изучении Python!
