Модуль 1: Основы Python

1.1 Установка и настройка Python

Сложность: начальная Время изучения: 15 мин

Введение

Python — это высокоуровневый язык программирования с простым и понятным синтаксисом, который делает его отличным выбором для начинающих. Благодаря своей универсальности Python используется в веб-разработке, анализе данных, искусственном интеллекте, автоматизации и многих других областях.

Прежде чем начать писать код, нам необходимо установить Python на компьютер. Процесс установки отличается в зависимости от операционной системы.

Установка Python

Windows

  1. Перейдите на официальный сайт Python: python.org/downloads
  2. Скачайте последнюю версию Python для Windows (на момент написания курса это Python 3.11)
  3. Запустите установщик и обязательно отметьте опцию "Add Python to PATH" — это очень важный шаг!
  4. Нажмите "Install Now" для стандартной установки
  5. После завершения установки нажмите "Close"

Важно: Опция "Add Python to PATH" позволит вам запускать Python из командной строки без указания полного пути. Если вы забыли отметить эту опцию при установке, вам придется либо переустановить Python, либо добавить его в PATH вручную.

# Проверка успешной установки в командной строке Windows
python --version
# Должно вывести что-то вроде: Python 3.11.2

macOS

В macOS Python обычно уже предустановлен, но это может быть устаревшая версия. Рекомендуется установить актуальную версию:

  1. Перейдите на официальный сайт Python: python.org/downloads
  2. Скачайте последнюю версию Python для macOS
  3. Откройте скачанный файл .pkg и следуйте инструкциям установщика
  4. После завершения установки нажмите "Close"
# Проверка успешной установки в терминале macOS
python3 --version
# Должно вывести что-то вроде: Python 3.11.2

Linux

В большинстве дистрибутивов Linux Python уже предустановлен. Проверьте версию командой:

# Проверка версии Python
python3 --version

Если Python не установлен или у вас устаревшая версия, используйте пакетный менеджер вашего дистрибутива:

# Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip

# Fedora
sudo dnf install python3 python3-pip

# Arch Linux
sudo pacman -S python python-pip

Первый запуск Python

После установки давайте убедимся, что Python работает корректно и напишем нашу первую программу!

Интерактивный режим

Python можно использовать в интерактивном режиме, что очень удобно для быстрых экспериментов и обучения:

# Запуск интерактивного режима
python     # В Windows
python3    # В macOS/Linux

Вы увидите приглашение командной строки Python (>>>). Теперь можно вводить команды:

# В интерактивном режиме
>>> print("Привет, мир!")
Привет, мир!
>>> 2 + 3
5
>>> exit()  # Выход из интерактивного режима

Запуск Python-скриптов

Для более сложных программ код обычно сохраняют в файлах с расширением .py:

  1. Создайте файл hello.py в любом текстовом редакторе
  2. Добавьте в него код: print("Привет, мир!")
  3. Сохраните файл
  4. Запустите скрипт из командной строки/терминала:
# Запуск Python-скрипта
python hello.py     # В Windows
python3 hello.py    # В macOS/Linux

Установка IDE (среды разработки)

Для удобства разработки рекомендуется установить интегрированную среду разработки (IDE) или текстовый редактор с поддержкой Python. Вот несколько популярных вариантов:

  • Visual Studio Code — бесплатный, легкий и мощный редактор кода с расширениями для Python. Идеален для начинающих благодаря простому интерфейсу и большому количеству обучающих материалов.
  • PyCharm — профессиональная IDE для Python с множеством функций. Имеет бесплатную Community версию и платную Professional версию с расширенными возможностями.
  • Jupyter Notebook — интерактивная среда для анализа данных и обучения. Позволяет комбинировать код, текст и визуализации в одном документе.

Установка Visual Studio Code и расширений для Python

  1. Скачайте и установите VS Code с официального сайта
  2. Запустите VS Code
  3. Перейдите в раздел расширений (Extensions) нажав Ctrl+Shift+X или кликнув на иконку в левой панели
  4. Найдите и установите расширение "Python" от Microsoft
  5. Также рекомендуется установить расширение "Pylance" для улучшенной поддержки языка

Совет: VS Code предлагает множество полезных функций для Python-разработчиков, включая подсветку синтаксиса, автодополнение кода, отладку, интеграцию с Git и многое другое. Потратьте немного времени на изучение этих возможностей — это значительно повысит вашу продуктивность!

Заключение

Поздравляем! Вы успешно установили Python и сделали первые шаги в программировании. Теперь вы готовы изучать синтаксис языка и писать более сложные программы.

Практическое задание: Первая программа

Создайте Python-скрипт hello_python.py, который выполняет следующие действия:

  1. Выводит приветствие "Привет, Python!"
  2. Выводит текущую версию Python (используйте модуль sys)
  3. Выводит текущую дату и время (используйте модуль datetime)
  4. Вычисляет и выводит результат математического выражения: (2023 - 1991) * 2 + 10
Подсказка

Для получения версии Python используйте sys.version, а для текущей даты - datetime.datetime.now(). Не забудьте импортировать необходимые модули в начале файла.

Решение 
import sys
import datetime

# Выводим приветствие
print("Привет, Python!")

# Выводим версию Python
print(f"Версия Python: {sys.version.split()[0]}")

# Выводим текущую дату и время
current_datetime = datetime.datetime.now()
formatted_date = current_datetime.strftime("%d.%m.%Y %H:%M:%S")
print(f"Текущая дата и время: {formatted_date}")

# Вычисляем и выводим результат выражения
result = (2023 - 1991) * 2 + 10
print(f"Результат выражения (2023 - 1991) * 2 + 10 = {result}")

# Дополнительно: проверяем, запущен ли скрипт напрямую
if __name__ == "__main__":
    print("Скрипт выполнен успешно!")

Этот скрипт демонстрирует базовые возможности Python: импорт модулей, вывод текста, форматирование строк, математические операции и использование условной конструкции if __name__ == "__main__", которая позволяет определить, запущен ли файл напрямую или импортирован как модуль.

1.2 Синтаксис и переменные

Сложность: начальная Время изучения: 20 мин

Основные элементы синтаксиса Python

Python славится своим чистым и читаемым синтаксисом, что делает его идеальным языком для начинающих программистов. В отличие от многих других языков, Python использует отступы для определения блоков кода, а не фигурные скобки или ключевые слова.

Отступы и блоки кода

В Python отступы имеют смысловое значение. Они используются для определения блоков кода в условиях, циклах, функциях и классах.

# Правильное использование отступов
if True:
    print("Этот код принадлежит блоку if")
    print("Этот код тоже в блоке if")
print("Этот код вне блока if")

# Неправильное использование отступов вызовет ошибку
if True:
print("Этот код вызовет ошибку IndentationError")

Важно: Стандарт PEP 8 рекомендует использовать 4 пробела для каждого уровня отступа. Избегайте смешивания табуляции и пробелов!

Комментарии

Комментарии используются для добавления пояснений к коду. Они игнорируются интерпретатором Python и не влияют на выполнение программы.

# Это однострочный комментарий

"""
Это многострочный комментарий
или строка документации (docstring)
Обычно используется для описания функций и классов
"""

print("Привет, мир!")  # Комментарий может быть и в конце строки кода

Переменные и присваивание

Переменные в Python — это имена, которые ссылаются на значения в памяти. В отличие от некоторых языков, в Python не нужно объявлять тип переменной — он определяется автоматически.

# Создание переменных и присваивание значений
name = "Алексей"      # строка
age = 25              # целое число
height = 1.85         # число с плавающей точкой
is_student = True     # логическое значение

# Вывод значений переменных
print(name)
print(age)
print(height)
print(is_student)

# Изменение значения переменной
age = 26
print("Новый возраст:", age)

Правила именования переменных

При выборе имен для переменных следуйте этим правилам:

  • Имя может содержать буквы (a-z, A-Z), цифры (0-9) и символ подчеркивания (_)
  • Имя не может начинаться с цифры
  • Имена чувствительны к регистру (name и Name — разные переменные)
  • Нельзя использовать зарезервированные слова Python (if, for, while и т.д.)
# Допустимые имена переменных
user_name = "John"
age1 = 30
_private = "Секрет"
firstName = "Иван"
CONSTANT = 3.14

# Недопустимые имена переменных
# 1name = "Ошибка"    # Начинается с цифры
# my-var = 10         # Содержит дефис
# if = "Ошибка"       # Зарезервированное слово

Соглашение: В Python принято использовать snake_case для имен переменных и функций (слова в нижнем регистре, разделенные подчеркиваниями). Для классов используется CamelCase (каждое слово с заглавной буквы без разделителей).

Множественное присваивание

Python позволяет присваивать значения нескольким переменным в одной строке:

# Присваивание одного значения нескольким переменным
x = y = z = 0
print(x, y, z)  # Выведет: 0 0 0

# Присваивание разных значений нескольким переменным
a, b, c = 1, 2, 3
print(a, b, c)  # Выведет: 1 2 3

# Обмен значениями переменных
a, b = b, a
print(a, b)     # Выведет: 2 1

Типы данных

Python имеет несколько встроенных типов данных. Тип переменной определяется значением, которое ей присвоено.

# Основные типы данных
number = 42             # int (целое число)
pi = 3.14159            # float (число с плавающей точкой)
name = "Python"         # str (строка)
is_active = True        # bool (логическое значение)
nothing = None          # NoneType (отсутствие значения)

# Проверка типа с помощью функции type()
print(type(number))     # 
print(type(pi))         # 
print(type(name))       # 
print(type(is_active))  # 
print(type(nothing))    #

Преобразование типов

Иногда необходимо преобразовать значение из одного типа в другой. Python предоставляет встроенные функции для этого:

# Преобразование типов
age_str = "25"
age_int = int(age_str)      # Преобразование строки в целое число
print(age_int + 5)          # 30

price = 19.99
price_int = int(price)      # Преобразование float в int (отбрасывает дробную часть)
print(price_int)            # 19

count = 100
count_str = str(count)      # Преобразование числа в строку
print("У вас " + count_str + " сообщений")  # "У вас 100 сообщений"

is_valid = 1
is_valid_bool = bool(is_valid)  # Преобразование в логическое значение
print(is_valid_bool)        # True (ненулевые числа преобразуются в True)

Примечание: Не все преобразования безопасны. Например, попытка преобразовать строку "привет" в число вызовет ошибку. Всегда проверяйте данные перед преобразованием.

Ввод и вывод данных

Функция print()

Для вывода данных в консоль используется функция print():

# Базовое использование print()
print("Привет, мир!")

# Вывод нескольких значений
name = "Анна"
age = 28
print("Имя:", name, "Возраст:", age)

# Форматирование строк (f-строки в Python 3.6+)
print(f"Привет, {name}! Тебе {age} лет.")

# Использование разделителя
print("Яблоко", "Банан", "Апельсин", sep=" | ")  # Яблоко | Банан | Апельсин

# Изменение символа конца строки
print("Строка 1", end=" --> ")
print("Строка 2")  # Строка 1 --> Строка 2

Функция input()

Для получения данных от пользователя используется функция input():

# Базовое использование input()
name = input("Введите ваше имя: ")
print(f"Привет, {name}!")

# Важно: input() всегда возвращает строку
age = input("Введите ваш возраст: ")
print(type(age))  # 

# Для использования в вычислениях нужно преобразовать в число
age_int = int(age)
next_year_age = age_int + 1
print(f"В следующем году вам будет {next_year_age} лет.")

Заключение

В этом разделе мы познакомились с основами синтаксиса Python и работой с переменными. Мы узнали о правилах именования переменных, типах данных и преобразовании типов, а также научились получать ввод от пользователя и выводить данные.

Практическое задание: Персональный калькулятор

Создайте программу-калькулятор, которая:

  1. Приветствует пользователя и представляется как "Персональный калькулятор"
  2. Запрашивает имя пользователя
  3. Запрашивает год рождения пользователя
  4. Вычисляет возраст пользователя в текущем году
  5. Предлагает пользователю ввести два числа
  6. Выполняет с этими числами основные арифметические операции (+, -, *, /)
  7. Выводит персонализированный отчет с результатами всех операций
Подсказка

Используйте функцию input() для получения данных, int() или float() для преобразования строк в числа. Для форматирования вывода используйте f-строки. Для получения текущего года используйте модуль datetime.

Решение 
import datetime

# Получение текущего года
current_year = datetime.datetime.now().year

# Приветствие
print("=" * 50)
print("Добро пожаловать в Персональный калькулятор!")
print("=" * 50)

# Запрос персональных данных
name = input("Как вас зовут? ")
try:
    birth_year = int(input(f"{name}, введите год вашего рождения: "))
    age = current_year - birth_year
    
    # Проверка корректности возраста
    if age < 0:
        print("Ошибка: Вы еще не родились! Используем возраст 0.")
        age = 0
    elif age > 120:
        print("Ошибка: Возраст слишком большой! Используем возраст 30.")
        age = 30
        
    # Запрос чисел для калькулятора
    print(f"\n{name}, теперь введите два числа для вычислений:")
    num1 = float(input("Первое число: "))
    num2 = float(input("Второе число: "))
    
    # Выполнение вычислений
    sum_result = num1 + num2
    diff_result = num1 - num2
    mult_result = num1 * num2
    
    # Проверка деления на ноль
    if num2 != 0:
        div_result = num1 / num2
        div_status = f"{num1} / {num2} = {div_result:.2f}"
    else:
        div_status = "Деление на ноль невозможно!"
    
    # Вывод персонализированного отчета
    print("\n" + "=" * 50)
    print(f"ОТЧЕТ ДЛЯ ПОЛЬЗОВАТЕЛЯ: {name.upper()}")
    print("=" * 50)
    print(f"Возраст: {age} лет")
    print(f"Год рождения: {birth_year}")
    print("\nРезультаты вычислений:")
    print(f"Сложение: {num1} + {num2} = {sum_result}")
    print(f"Вычитание: {num1} - {num2} = {diff_result}")
    print(f"Умножение: {num1} * {num2} = {mult_result}")
    print(f"Деление: {div_status}")
    print("=" * 50)
    print(f"Спасибо за использование Персонального калькулятора, {name}!")
    
except ValueError:
    print("Ошибка: Введены некорректные данные. Пожалуйста, запустите программу снова.")

Эта программа демонстрирует работу с вводом пользователя, преобразование типов, математические операции, форматирование вывода и обработку ошибок с помощью конструкции try-except. Также программа проверяет корректность введенных данных, что является хорошей практикой при разработке.

1.3 Типы данных

Сложность: начальная Время изучения: 25 мин

Основные типы данных в Python

Python имеет несколько встроенных типов данных, которые позволяют хранить и обрабатывать различные виды информации. Понимание типов данных — важный фундамент для эффективного программирования.

В этом разделе мы рассмотрим основные типы данных, с которыми вы будете работать чаще всего.

Числовые типы

Python поддерживает несколько числовых типов данных:

  • int (целые числа) — числа без дробной части: 1, 100, -10, 0
  • float (числа с плавающей точкой) — числа с дробной частью: 3.14, -0.001, 2.0
  • complex (комплексные числа) — числа с действительной и мнимой частью: 3+5j, 1-2j
# Примеры числовых типов
a = 10          # int
b = 3.14        # float
c = 2 + 3j      # complex

# Проверка типа
print(type(a))  # 
print(type(b))  # 
print(type(c))  # 

# Математические операции
print(a + 5)    # 15
print(b * 2)    # 6.28
print(c.real)   # 2.0 (действительная часть)
print(c.imag)   # 3.0 (мнимая часть)

Интересный факт: В Python целые числа (int) могут быть произвольной длины, ограниченной только доступной памятью. Это позволяет работать с очень большими числами без потери точности.

Строковый тип (str)

Строки в Python представляют собой последовательности символов, заключенные в кавычки. Они используются для хранения текстовой информации.

# Способы создания строк
s1 = 'Привет'        # одинарные кавычки
s2 = "Мир"           # двойные кавычки
s3 = '''Многострочная
строка'''            # тройные кавычки для многострочного текста

# Базовые операции со строками
print(s1 + ' ' + s2)  # конкатенация (объединение): "Привет Мир"
print(s1 * 3)         # повторение: "ПриветПриветПривет"
print(len(s1))        # длина строки: 6
print(s1[0])          # доступ по индексу: "П" (индексы начинаются с 0)
print(s1[1:4])        # срезы: "рив" (символы с индекса 1 до 3 включительно)

# Полезные методы строк
text = "Python - отличный язык программирования"
print(text.upper())           # "PYTHON - ОТЛИЧНЫЙ ЯЗЫК ПРОГРАММИРОВАНИЯ"
print(text.lower())           # "python - отличный язык программирования"
print(text.replace("отличный", "замечательный"))  # замена подстроки
print(text.split())           # разделение на список слов: ['Python', '-', 'отличный', 'язык', 'программирования']
print("   текст   ".strip())  # удаление пробелов в начале и конце: "текст"

Строки в Python являются неизменяемыми (иммутабельными). Это означает, что после создания строки вы не можете изменить её содержимое. Вместо этого при любой модификации создаётся новая строка.

Логический тип (bool)

Тип bool может принимать только два значения: True (истина) и False (ложь). Логические значения используются для управления потоком выполнения программы и проверки условий.

# Создание логических значений
is_active = True
is_completed = False

# Логические операции
print(is_active and is_completed)  # False (И: оба значения должны быть True)
print(is_active or is_completed)   # True (ИЛИ: хотя бы один операнд должен быть True)
print(not is_active)               # False (НЕ: инвертирует значение)

# Сравнения возвращают логические значения
x = 10
y = 5
print(x > y)    # True
print(x == y)   # False
print(x != y)   # True

# Проверка истинности значений
# В Python любое ненулевое число или непустая строка/список считается True
print(bool(0))      # False
print(bool(1))      # True
print(bool(""))     # False
print(bool("abc"))  # True
print(bool([]))     # False (пустой список)
print(bool([1,2]))  # True (непустой список)

Тип None

None — это специальное значение, которое представляет отсутствие значения или нулевой указатель. Часто используется как значение по умолчанию для функций или для инициализации переменных.

# Использование None
result = None
print(result)        # None
print(type(result))  # 

# Проверка на None
if result is None:
    print("Значение не задано")
    
# Функция, возвращающая None
def function_without_return():
    pass  # ничего не делает

x = function_without_return()
print(x)  # None

Важно: Для проверки на None всегда используйте операторы is и is not, а не операторы сравнения == и !=.

Преобразование типов

Python позволяет преобразовывать данные из одного типа в другой с помощью встроенных функций:

# Преобразование в целое число (int)
print(int(3.14))      # 3 (отбрасывает дробную часть)
print(int("42"))      # 42 (из строки)
print(int(True))      # 1
print(int(False))     # 0

# Преобразование в число с плавающей точкой (float)
print(float(5))       # 5.0
print(float("3.14"))  # 3.14
print(float(True))    # 1.0

# Преобразование в строку (str)
print(str(42))        # "42"
print(str(3.14))      # "3.14"
print(str(True))      # "True"

# Преобразование в логическое значение (bool)
print(bool(1))        # True
print(bool(0))        # False
print(bool(""))       # False (пустая строка)
print(bool("False"))  # True (непустая строка)

1.4 Базовые операторы

Сложность: начальная Время изучения: 20 мин

Операторы в Python

Операторы — это специальные символы или ключевые слова, которые выполняют операции над переменными и значениями. Python предоставляет широкий набор операторов для различных типов операций.

Арифметические операторы

Арифметические операторы используются для выполнения математических операций:

# Основные арифметические операторы
a = 10
b = 3

print(a + b)    # 13 (сложение)
print(a - b)    # 7 (вычитание)
print(a * b)    # 30 (умножение)
print(a / b)    # 3.3333... (деление с плавающей точкой)
print(a // b)   # 3 (целочисленное деление, отбрасывает дробную часть)
print(a % b)    # 1 (остаток от деления)
print(a ** b)   # 1000 (возведение в степень: 10^3)

# Примеры с отрицательными числами
print(5 % -3)   # 2 (знак берется от делителя)
print(-5 % 3)   # 1 (результат всегда меньше делителя и >= 0)

# Округление результатов
print(round(10 / 3, 2))  # 3.33 (округление до 2 знаков после запятой)

Операторы сравнения

Операторы сравнения используются для сравнения значений. Они возвращают логические значения True или False:

# Операторы сравнения
x = 5
y = 10

print(x == y)   # False (равно)
print(x != y)   # True (не равно)
print(x > y)    # False (больше)
print(x < y)    # True (меньше)
print(x >= y)   # False (больше или равно)
print(x <= y)   # True (меньше или равно)

# Сравнение строк (по алфавиту)
print("apple" < "banana")  # True
print("apple" < "Apple")   # False (заглавные буквы идут перед строчными в ASCII)

# Сравнение разных типов
print(5 == "5")   # False (разные типы)
print(5 == 5.0)   # True (Python автоматически приводит типы)

# Цепочки сравнений
age = 25
print(18 <= age < 65)  # True (возраст между 18 и 64 включительно)

Логические операторы

Логические операторы используются для комбинирования условных выражений:

# Логические операторы
p = True
q = False

print(p and q)  # False (логическое И: оба операнда должны быть True)
print(p or q)   # True (логическое ИЛИ: хотя бы один операнд должен быть True)
print(not p)    # False (логическое НЕ: инвертирует значение)

# Примеры с условиями
age = 25
has_license = True
print(age >= 18 and has_license)  # True (может водить машину)

temperature = 15
is_raining = True
print(temperature > 20 or not is_raining)  # False (не хорошая погода для прогулки)

# Короткое замыкание (short-circuit evaluation)
# Если первый операнд and равен False, второй не вычисляется
print(False and print("Этот код не выполнится"))  # False

# Если первый операнд or равен True, второй не вычисляется
print(True or print("Этот код не выполнится"))    # True

Операторы присваивания

Операторы присваивания используются для присваивания значений переменным:

# Базовое присваивание
n = 10

# Составные операторы присваивания
n += 5          # эквивалентно n = n + 5, результат: 15
n -= 2          # эквивалентно n = n - 2, результат: 13
n *= 3          # эквивалентно n = n * 3, результат: 39
n /= 3          # эквивалентно n = n / 3, результат: 13.0
n //= 2         # эквивалентно n = n // 2, результат: 6.0
n %= 4          # эквивалентно n = n % 4, результат: 2.0
n **= 3         # эквивалентно n = n ** 3, результат: 8.0

# Присваивание со строками
text = "Hello"
text += " World"  # эквивалентно text = text + " World"
print(text)       # "Hello World"

Операторы принадлежности (in, not in)

Операторы принадлежности проверяют, содержится ли элемент в последовательности (строке, списке и т.д.):

# Проверка наличия символа в строке
text = "Python Programming"
print('P' in text)       # True
print('z' in text)       # False
print('z' not in text)   # True

# Проверка наличия подстроки
print("thon" in text)    # True
print("Java" in text)    # False

# Проверка наличия элемента в списке
languages = ["Python", "Java", "C++", "JavaScript"]
print("Python" in languages)     # True
print("Ruby" not in languages)   # True

Операторы тождественности (is, is not)

Операторы тождественности проверяют, являются ли два объекта одним и тем же объектом в памяти:

# Операторы тождественности
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)    # True (содержимое одинаковое)
print(a is b)    # False (разные объекты в памяти)
print(a is c)    # True (один и тот же объект)

# Проверка на None
x = None
print(x is None)       # True (правильный способ проверки на None)
print(x is not None)   # False

Важно: Используйте == для сравнения значений и is для проверки тождественности объектов. Для проверки на None всегда используйте is None, а не == None.

Приоритет операторов

В Python, как и в математике, операторы имеют разный приоритет, который определяет порядок их выполнения. Вот основные правила приоритета (от высшего к низшему):

  1. Скобки () — всегда выполняются первыми
  2. Возведение в степень **
  3. Унарные операторы +x, -x, ~x
  4. Умножение, деление *, /, //, %
  5. Сложение, вычитание +, -
  6. Операторы сравнения ==, !=, >, >=, <, <=
  7. Логические операторы not, and, or
# Примеры приоритета операторов
print(2 + 3 * 4)     # 14 (сначала умножение, потом сложение)
print((2 + 3) * 4)   # 20 (сначала выражение в скобках, потом умножение)
print(10 / 2 + 3)    # 8.0 (сначала деление, потом сложение)
print(10 / (2 + 3))  # 2.0 (сначала выражение в скобках, потом деление)

# Логические операторы
print(True or False and False)  # True (and имеет более высокий приоритет, чем or)
print((True or False) and False)  # False (скобки изменяют порядок выполнения)

# Если сомневаетесь, используйте скобки для явного указания порядка операций
complex_expression = (10 + 5) * ((7 - 3) / 2)
print(complex_expression)  # 30.0

В этом разделе мы познакомились с основными операторами Python: арифметическими, сравнения, логическими, присваивания, принадлежности и тождественности. Также мы узнали о приоритете операторов, который определяет порядок их выполнения.

Понимание операторов и их приоритета — важный навык для написания корректных выражений и условий в программах.

Практическое задание: Интерактивный калькулятор с операторами

Создайте интерактивный калькулятор, который:

  1. Приветствует пользователя и показывает меню доступных операций
  2. Позволяет пользователю выбрать операцию из меню (сложение, вычитание, умножение, деление, возведение в степень, остаток от деления)
  3. Запрашивает два числа для выполнения выбранной операции
  4. Выполняет операцию и показывает результат
  5. Спрашивает, хочет ли пользователь выполнить еще одну операцию
  6. Использует различные операторы сравнения и логические операторы для проверки ввода
Подсказка

Используйте цикл while для повторения операций, словарь для хранения доступных операций, условные операторы для проверки ввода. Не забудьте обработать случаи деления на ноль и некорректного ввода.

Решение 
def main():
    print("=" * 50)
    print("ИНТЕРАКТИВНЫЙ КАЛЬКУЛЯТОР")
    print("=" * 50)
    print("Добро пожаловать в интерактивный калькулятор!")
    
    # Определяем доступные операции
    operations = {
        '1': ('Сложение', lambda x, y: x + y),
        '2': ('Вычитание', lambda x, y: x - y),
        '3': ('Умножение', lambda x, y: x * y),
        '4': ('Деление', lambda x, y: x / y if y != 0 else "Ошибка: деление на ноль"),
        '5': ('Возведение в степень', lambda x, y: x ** y),
        '6': ('Остаток от деления', lambda x, y: x % y if y != 0 else "Ошибка: деление на ноль"),
        '7': ('Целочисленное деление', lambda x, y: x // y if y != 0 else "Ошибка: деление на ноль")
    }
    
    continue_calc = True
    
    while continue_calc:
        # Показываем меню
        print("\nДоступные операции:")
        for key, (name, _) in operations.items():
            print(f"{key}. {name}")
        print("0. Выход")
        
        # Получаем выбор пользователя
        choice = input("\nВыберите операцию (0-7): ")
        
        # Проверяем выбор
        if choice == '0':
            print("Спасибо за использование калькулятора! До свидания!")
            break
        
        if choice not in operations:
            print("Ошибка: Неверный выбор операции. Пожалуйста, выберите от 0 до 7.")
            continue
        
        # Получаем название и функцию выбранной операции
        operation_name, operation_func = operations[choice]
        
        # Запрашиваем числа
        try:
            num1 = float(input("Введите первое число: "))
            num2 = float(input("Введите второе число: "))
            
            # Выполняем операцию
            result = operation_func(num1, num2)
            
            # Проверяем результат на ошибку
            if isinstance(result, str):
                print(result)
            else:
                # Форматируем вывод в зависимости от результата
                if result.is_integer():
                    result = int(result)
                
                print(f"\nРезультат операции '{operation_name}':")
                print(f"{num1} {get_operation_symbol(choice)} {num2} = {result}")
                
                # Дополнительная информация о результате
                if result > 0:
                    print("Результат положительный")
                elif result < 0:
                    print("Результат отрицательный")
                else:
                    print("Результат равен нулю")
                
                if isinstance(result, int) and result != 0:
                    if result % 2 == 0:
                        print("Результат - четное число")
                    else:
                        print("Результат - нечетное число")
        
        except ValueError:
            print("Ошибка: Введены некорректные числа. Пожалуйста, попробуйте снова.")
            continue
        
        # Спрашиваем, хочет ли пользователь продолжить
        continue_choice = input("\nХотите выполнить еще одну операцию? (да/нет): ").lower()
        continue_calc = continue_choice == 'да' or continue_choice == 'y' or continue_choice == 'yes'

def get_operation_symbol(choice):
    symbols = {
        '1': '+',
        '2': '-',
        '3': '*',
        '4': '/',
        '5': '**',
        '6': '%',
        '7': '//'
    }
    return symbols.get(choice, '?')
