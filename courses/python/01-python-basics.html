1.1 Установка и настройка Python

Сложность: начальная Время изучения: 15 мин

Введение

Python — высокоуровневый язык программирования с простым и понятным синтаксисом. Он широко используется в веб-разработке, анализе данных, машинном обучении и автоматизации.

Перед началом программирования нужно настроить среду разработки.

Установка Python

Windows

  1. Скачайте Python с python.org/downloads
  2. Запустите инсталлятор и обязательно отметьте "Add Python to PATH"
  3. Нажмите "Install Now"

Важно: Опция "Add Python to PATH" позволяет запускать Python из командной строки.

# Верификация успешной установки в командной строке Windows
python --version
# Ожидаемый результат: Python 3.11.x

macOS

В macOS Python может быть уже установлен, но лучше установить новую версию:

  1. Скачайте Python с python.org/downloads
  2. Откройте скачанный .pkg файл и следуйте инструкциям
# Верификация успешной установки в терминале macOS
python3 --version
# Ожидаемый результат: Python 3.11.x

Linux

В большинстве дистрибутивов Linux Python уже установлен. Проверьте версию:

# Проверка версии Python
python3 --version

Если Python отсутствует или требуется обновление, используйте пакетный менеджер вашего дистрибутива:

# Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip

# Fedora
sudo dnf install python3 python3-pip

# Arch Linux
sudo pacman -S python python-pip

Первый запуск Python

После установки проверим работу Python и создадим первую программу.

Интерактивный режим

Python имеет интерактивный режим для быстрого тестирования кода:

# Запуск интерактивного режима
python     # В Windows
python3    # В macOS/Linux

После запуска вы увидите приглашение интерпретатора Python (>>>), где можно вводить и исполнять команды непосредственно:

# В интерактивном режиме
>>> print("Привет, мир!")
Привет, мир!
>>> 2 + 3
5
>>> exit()  # Завершение сеанса интерактивного режима

Запуск Python-скриптов

Для разработки полноценных программ код сохраняется в файлах с расширением .py:

  1. Создайте файл hello.py в текстовом редакторе
  2. Добавьте следующий код: print("Привет, мир!")
  3. Сохраните файл
  4. Выполните скрипт через командную строку или терминал:
# Запуск Python-скрипта
python hello.py     # В Windows
python3 hello.py    # В macOS/Linux

Установка IDE

Для удобной разработки рекомендуется использовать специальную среду:

  • Visual Studio Code — современный редактор с поддержкой Python. Идеален для начинающих.
  • PyCharm — профессиональная среда для Python-разработки.
  • Jupyter Notebook — интерактивная среда для анализа данных и обучения.

Настройка Visual Studio Code для Python

  1. Установите VS Code с официального сайта
  2. Откройте VS Code и нажмите Ctrl+Shift+X
  3. Найдите и установите расширение "Python" от Microsoft
  4. Дополнительно установите "Pylance" для улучшенной поддержки языка

Профессиональный совет: VS Code предоставляет множество инструментов для повышения продуктивности Python-разработчика, включая интеллектуальное автодополнение кода, встроенный отладчик, интеграцию с системами контроля версий, линтеры и форматтеры кода. Инвестирование времени в изучение этих возможностей существенно ускорит ваш рабочий процесс и повысит качество кода.

Заключение

Поздравляем! Вы настроили среду разработки Python и готовы к изучению языка. Теперь вы готовы к изучению синтаксиса языка и созданию более сложных программ.

Практическое задание: Первая программа

Разработайте Python-скрипт hello_python.py, реализующий следующую функциональность:

  1. Вывод приветственного сообщения "Привет, Python!"
  2. Определение и вывод текущей версии Python (с использованием модуля sys)
  3. Определение и вывод текущей даты и времени (с использованием модуля datetime)
  4. Вычисление и вывод результата математического выражения: (2023 - 1991) * 2 + 10
Подсказка

Для получения информации о версии Python используйте атрибут sys.version, а для работы с текущей датой и временем — метод datetime.datetime.now(). Не забудьте импортировать соответствующие модули в начале файла.

Решение 
import sys
import datetime

# Выводим приветствие
print("Привет, Python!")

# Выводим версию Python
print(f"Версия Python: {sys.version.split()[0]}")

# Выводим текущую дату и время
current_datetime = datetime.datetime.now()
formatted_date = current_datetime.strftime("%d.%m.%Y %H:%M:%S")
print(f"Текущая дата и время: {formatted_date}")

# Вычисляем и выводим результат выражения
result = (2023 - 1991) * 2 + 10
print(f"Результат выражения (2023 - 1991) * 2 + 10 = {result}")

# Дополнительно: проверяем, запущен ли скрипт напрямую
if __name__ == "__main__":
    print("Скрипт выполнен успешно!")

Этот скрипт демонстрирует базовые возможности Python: импорт модулей, вывод текста, форматирование строк, математические операции и использование условной конструкции if __name__ == "__main__", которая позволяет определить, запущен ли файл напрямую или импортирован как модуль.

1.2 Синтаксис и переменные

Сложность: начальная Время изучения: 20 мин

Основные элементы синтаксиса Python

Python имеет простой и читаемый синтаксис. Вместо фигурных скобок Python использует отступы для выделения блоков кода.

Отступы и блоки кода

В Python отступы определяют структуру кода. Они показывают, какие строки относятся к блоку условия, цикла или функции.

# Корректное использование отступов
if True:
    print("Данный код относится к блоку условного оператора if")
    print("Этот код также находится в блоке if")
print("Этот код находится вне блока if")

# Некорректное использование отступов приведет к ошибке
if True:
print("Этот код вызовет ошибку IndentationError")

Важно: Согласно стандарту PEP 8 (руководство по стилю кода Python), рекомендуется использовать 4 пробела для каждого уровня отступа. Категорически не рекомендуется смешивать табуляцию и пробелы в одном проекте!

Комментарии

Комментарии — важный инструмент документирования кода. Они игнорируются интерпретатором Python и предназначены исключительно для разработчиков.

# Это однострочный комментарий

"""
Это многострочный комментарий
или документационная строка (docstring)
Обычно используется для документирования функций, классов и модулей
"""

print("Привет, мир!")  # Комментарий может располагаться после кода в той же строке

Переменные и присваивание

Переменные в Python — это имена, которые указывают на данные в памяти. Тип переменной определяется автоматически при присваивании значения.

# Создание переменных и присваивание значений
name = "Алексей"      # строковый тип (str)
age = 25              # целочисленный тип (int)
height = 1.85         # тип с плавающей точкой (float)
is_student = True     # логический тип (bool)

# Вывод значений переменных
print(name)
print(age)
print(height)
print(is_student)

# Модификация значения переменной
age = 26
print(f"Обновленный возраст: {age}")

Правила именования переменных

При выборе имен для переменных следует руководствоваться следующими правилами:

  • Имя может содержать буквы латинского алфавита (a-z, A-Z), цифры (0-9) и символ подчеркивания (_)
  • Имя не может начинаться с цифры
  • Имена чувствительны к регистру (name и Name интерпретируются как разные переменные)
  • Запрещено использовать зарезервированные слова Python (if, for, while, class, def и т.д.)
# Допустимые имена переменных
user_name = "John"
age1 = 30
_private = "Секретные данные"
firstName = "Иван"
CONSTANT = 3.14159

# Недопустимые имена переменных
# 1name = "Ошибка"    # Начинается с цифры
# my-var = 10         # Содержит недопустимый символ (дефис)
# if = "Ошибка"       # Использует зарезервированное слово

Соглашение о стиле: В Python принято использовать snake_case для именования переменных и функций (слова в нижнем регистре, разделенные подчеркиваниями). Для классов используется стиль CamelCase (каждое слово начинается с заглавной буквы, без разделителей). Константы обычно записываются в верхнем регистре с подчеркиваниями (например, MAX_VALUE).

Множественное присваивание

Python предоставляет элегантный синтаксис для одновременного присваивания значений нескольким переменным:

# Присваивание одного значения нескольким переменным
x = y = z = 0
print(x, y, z)  # Результат: 0 0 0

# Присваивание разных значений нескольким переменным
a, b, c = 1, 2, 3
print(a, b, c)  # Результат: 1 2 3

# Обмен значениями переменных (без использования временной переменной)
a, b = b, a
print(a, b)     # Результат: 2 1

Типы данных

Python обладает богатой системой встроенных типов данных. Тип переменной определяется автоматически при присваивании значения и может изменяться в процессе выполнения программы.

# Основные типы данных
number = 42             # int (целое число)
pi = 3.14159            # float (число с плавающей точкой)
name = "Python"         # str (строка)
is_active = True        # bool (логическое значение)
nothing = None          # NoneType (отсутствие значения)

# Определение типа с помощью функции type()
print(type(number))     # 
print(type(pi))         # 
print(type(name))       # 
print(type(is_active))  # 
print(type(nothing))    #

Преобразование типов

В определенных ситуациях необходимо явное преобразование данных из одного типа в другой. Python предоставляет встроенные функции для таких преобразований:

# Преобразование типов
age_str = "25"
age_int = int(age_str)      # Преобразование строки в целое число
print(age_int + 5)          # Результат: 30

price = 19.99
price_int = int(price)      # Преобразование float в int (отбрасывает дробную часть)
print(price_int)            # Результат: 19

count = 100
count_str = str(count)      # Преобразование числа в строку
print("У вас " + count_str + " непрочитанных сообщений")  # "У вас 100 непрочитанных сообщений"

is_valid = 1
is_valid_bool = bool(is_valid)  # Преобразование в логическое значение
print(is_valid_bool)        # Результат: True (ненулевые числа преобразуются в True)

Примечание по безопасности: Не все преобразования типов выполняются безопасно. Например, попытка преобразовать строку "привет" в число вызовет исключение ValueError. Рекомендуется всегда проверять данные перед преобразованием или использовать обработку исключений.

Ввод и вывод данных

Функция print()

Используется для вывода информации в консоль:

# Базовое использование print()
print("Привет, мир!")

# Вывод нескольких значений
name = "Анна"
age = 28
print("Имя:", name, "Возраст:", age)

# Форматирование строк с использованием f-строк (Python 3.6+)
print(f"Привет, {name}! Вам {age} лет.")

# Использование разделителя между аргументами
print("Яблоко", "Банан", "Апельсин", sep=" | ")  # Результат: Яблоко | Банан | Апельсин

# Модификация символа окончания строки
print("Строка 1", end=" --> ")
print("Строка 2")  # Результат: Строка 1 --> Строка 2

Функция input()

Получает данные от пользователя:

# Базовое использование input()
name = input("Введите ваше имя: ")
print(f"Здравствуйте, {name}!")

# Важно: input() всегда возвращает данные в виде строки
age = input("Введите ваш возраст: ")
print(type(age))  # 

# Для использования в арифметических операциях необходимо преобразование в числовой тип
age_int = int(age)
next_year_age = age_int + 1
print(f"В следующем году вам исполнится {next_year_age} лет.")

1.3 Типы данных

Сложность: начальная Время изучения: 25 мин

Основные типы данных в Python

Python имеет разнообразные встроенные типы данных для работы с разными видами информации.

Числовые типы

Python поддерживает несколько числовых типов:

  • int (целые числа) — числа без дробной части, произвольной точности: 1, 100, -10, 0
  • float (числа с плавающей точкой) — числа с дробной частью, соответствующие стандарту IEEE 754: 3.14, -0.001, 2.0
  • complex (комплексные числа) — числа с действительной и мнимой частями: 3+5j, 1-2j
# Примеры числовых типов
a = 10          # int (целое число)
b = 3.14        # float (число с плавающей точкой)
c = 2 + 3j      # complex (комплексное число)

# Определение типа
print(type(a))  # 
print(type(b))  # 
print(type(c))  # 

# Математические операции
print(a + 5)    # 15
print(b * 2)    # 6.28
print(c.real)   # 2.0 (действительная часть)
print(c.imag)   # 3.0 (мнимая часть)

Особенность: В Python целые числа имеют неограниченную точность, что позволяет работать с очень большими числами.

Строковый тип (str)

Строки — это последовательности символов, заключенные в кавычки.

# Способы объявления строк
s1 = 'Привет'        # одинарные кавычки
s2 = "Мир"           # двойные кавычки
s3 = '''Многострочный
текст'''             # тройные кавычки для многострочного содержимого

# Основные операции со строками
print(s1 + ' ' + s2)  # конкатенация (объединение): "Привет Мир"
print(s1 * 3)         # повторение: "ПриветПриветПривет"
print(len(s1))        # определение длины: 6 символов
print(s1[0])          # доступ по индексу: "П" (индексация начинается с 0)
print(s1[1:4])        # срезы (slicing): "рив" (символы с индекса 1 по 3 включительно)

# Полезные методы для работы со строками
text = "Python - мощный язык программирования"
print(text.upper())           # "PYTHON - МОЩНЫЙ ЯЗЫК ПРОГРАММИРОВАНИЯ"
print(text.lower())           # "python - мощный язык программирования"
print(text.replace("мощный", "элегантный"))  # замена подстроки
print(text.split())           # разделение на список слов: ['Python', '-', 'мощный', 'язык', 'программирования']
print("   избыточные пробелы   ".strip())  # удаление начальных и конечных пробелов

Строки в Python неизменяемы. После создания строки её нельзя изменить. Операции со строками создают новые строки.

Логический тип (bool)

Тип bool имеет два значения: True и False. Используется в условиях и логических операциях.

# Объявление логических переменных
is_active = True
is_completed = False

# Логические операции
print(is_active and is_completed)  # False (логическое И: требует истинности обоих операндов)
print(is_active or is_completed)   # True (логическое ИЛИ: требует истинности хотя бы одного операнда)
print(not is_active)               # False (логическое НЕ: инвертирует значение)

# Операции сравнения возвращают логические значения
x = 10
y = 5
print(x > y)    # True (больше)
print(x == y)   # False (равно)
print(x != y)   # True (не равно)

# Правила приведения к логическому типу
# В Python любое ненулевое число или непустая коллекция считается истиной (True)
print(bool(0))      # False
print(bool(42))     # True
print(bool(""))     # False (пустая строка)
print(bool("abc"))  # True (непустая строка)
print(bool([]))     # False (пустой список)
print(bool([1,2]))  # True (непустой список)

Тип None

None означает отсутствие значения. Часто используется как значение по умолчанию для параметров функций.

# Использование None
result = None
print(result)        # None
print(type(result))  # 

# Проверка на None
if result is None:
    print("Значение не определено")
    
# Функция без явного оператора return возвращает None
def function_without_return():
    pass  # пустая операция

x = function_without_return()
print(x)  # None

Совет: Для проверки на None используйте is и is not, а не == и !=.

Преобразование типов

Python предоставляет функции для преобразования данных между типами:

# Преобразование в целое число (int)
print(int(3.14))      # 3 (отбрасывает дробную часть без округления)
print(int("42"))      # 42 (преобразование из строки)
print(int(True))      # 1 (True соответствует 1)
print(int(False))     # 0 (False соответствует 0)

# Преобразование в число с плавающей точкой (float)
print(float(5))       # 5.0
print(float("3.14"))  # 3.14
print(float(True))    # 1.0

# Преобразование в строку (str)
print(str(42))        # "42"
print(str(3.14))      # "3.14"
print(str(True))      # "True"

# Преобразование в логическое значение (bool)
print(bool(1))        # True
print(bool(0))        # False
print(bool(""))       # False (пустая строка)
print(bool("False"))  # True (непустая строка, даже содержащая слово "False")

1.4 Базовые операторы

Сложность: начальная Время изучения: 20 мин

Операторы в Python

Операторы выполняют действия над переменными и значениями.

Арифметические операторы

Используются для математических операций:

# Основные арифметические операторы
a = 10
b = 3

print(a + b)    # 13 (сложение)
print(a - b)    # 7 (вычитание)
print(a * b)    # 30 (умножение)
print(a / b)    # 3.3333... (деление с плавающей точкой)
print(a // b)   # 3 (целочисленное деление, отбрасывает дробную часть)
print(a % b)    # 1 (остаток от деления)
print(a ** b)   # 1000 (возведение в степень: 10^3)

# Примеры с отрицательными числами
print(5 % -3)   # 2 (знак берется от делителя)
print(-5 % 3)   # 1 (результат всегда меньше делителя и >= 0)

# Округление результатов
print(round(10 / 3, 2))  # 3.33 (округление до 2 знаков после запятой)

Операторы сравнения

Сравнивают значения и возвращают True или False:

# Операторы сравнения
x = 5
y = 10

print(x == y)   # False (равно)
print(x != y)   # True (не равно)
print(x > y)    # False (больше)
print(x < y)    # True (меньше)
print(x >= y)   # False (больше или равно)
print(x <= y)   # True (меньше или равно)

# Сравнение строк (по алфавиту)
print("apple" < "banana")  # True
print("apple" < "Apple")   # False (заглавные буквы идут перед строчными в ASCII)

# Сравнение разных типов
print(5 == "5")   # False (разные типы)
print(5 == 5.0)   # True (Python автоматически приводит типы)

# Цепочки сравнений
age = 25
print(18 <= age < 65)  # True (возраст между 18 и 64 включительно)

Логические операторы

Логические операторы используются для комбинирования условных выражений:

# Логические операторы
p = True
q = False

print(p and q)  # False (логическое И: оба операнда должны быть True)
print(p or q)   # True (логическое ИЛИ: хотя бы один операнд должен быть True)
print(not p)    # False (логическое НЕ: инвертирует значение)

# Примеры с условиями
age = 25
has_license = True
print(age >= 18 and has_license)  # True (может водить машину)

temperature = 15
is_raining = True
print(temperature > 20 or not is_raining)  # False (не хорошая погода для прогулки)

# Короткое замыкание (short-circuit evaluation)
# Если первый операнд and равен False, второй не вычисляется
print(False and print("Этот код не выполнится"))  # False

# Если первый операнд or равен True, второй не вычисляется
print(True or print("Этот код не выполнится"))    # True

Операторы присваивания

Операторы присваивания используются для присваивания значений переменным:

# Базовое присваивание
n = 10

# Составные операторы присваивания
n += 5          # эквивалентно n = n + 5, результат: 15
n -= 2          # эквивалентно n = n - 2, результат: 13
n *= 3          # эквивалентно n = n * 3, результат: 39
n /= 3          # эквивалентно n = n / 3, результат: 13.0
n //= 2         # эквивалентно n = n // 2, результат: 6.0
n %= 4          # эквивалентно n = n % 4, результат: 2.0
n **= 3         # эквивалентно n = n ** 3, результат: 8.0

# Присваивание со строками
text = "Hello"
text += " World"  # эквивалентно text = text + " World"
print(text)       # "Hello World"

Операторы принадлежности (in, not in)

Операторы принадлежности проверяют, содержится ли элемент в последовательности (строке, списке и т.д.):

# Проверка наличия символа в строке
text = "Python Programming"
print('P' in text)       # True
print('z' in text)       # False
print('z' not in text)   # True

# Проверка наличия подстроки
print("thon" in text)    # True
print("Java" in text)    # False

# Проверка наличия элемента в списке
languages = ["Python", "Java", "C++", "JavaScript"]
print("Python" in languages)     # True
print("Ruby" not in languages)   # True

Операторы тождественности (is, is not)

Операторы тождественности проверяют, являются ли два объекта одним и тем же объектом в памяти:

# Операторы тождественности
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)    # True (содержимое одинаковое)
print(a is b)    # False (разные объекты в памяти)
print(a is c)    # True (один и тот же объект)

# Проверка на None
x = None
print(x is None)       # True (правильный способ проверки на None)
print(x is not None)   # False

Важно: Используйте == для сравнения значений и is для проверки тождественности объектов. Для проверки на None всегда используйте is None, а не == None.

Приоритет операторов

В Python, как и в математике, операторы имеют разный приоритет, который определяет порядок их выполнения. Вот основные правила приоритета (от высшего к низшему):

  1. Скобки () — всегда выполняются первыми
  2. Возведение в степень **
  3. Унарные операторы +x, -x, ~x
  4. Умножение, деление *, /, //, %
  5. Сложение, вычитание +, -
  6. Операторы сравнения ==, !=, >, >=, <, <=
  7. Логические операторы not, and, or
# Примеры приоритета операторов
print(2 + 3 * 4)     # 14 (сначала умножение, потом сложение)
print((2 + 3) * 4)   # 20 (сначала выражение в скобках, потом умножение)
print(10 / 2 + 3)    # 8.0 (сначала деление, потом сложение)
print(10 / (2 + 3))  # 2.0 (сначала выражение в скобках, потом деление)

# Логические операторы
print(True or False and False)  # True (and имеет более высокий приоритет, чем or)
print((True or False) and False)  # False (скобки изменяют порядок выполнения)

# Если сомневаетесь, используйте скобки для явного указания порядка операций
complex_expression = (10 + 5) * ((7 - 3) / 2)
print(complex_expression)  # 30.0

Понимание операторов и их приоритета — важный навык для написания корректных выражений и условий в программах.

Практическое задание: Интерактивный калькулятор с операторами

Создайте интерактивный калькулятор, который:

  1. Приветствует пользователя и показывает меню доступных операций
  2. Позволяет пользователю выбрать операцию из меню (сложение, вычитание, умножение, деление, возведение в степень, остаток от деления)
  3. Запрашивает два числа для выполнения выбранной операции
  4. Выполняет операцию и показывает результат
  5. Спрашивает, хочет ли пользователь выполнить еще одну операцию
  6. Использует различные операторы сравнения и логические операторы для проверки ввода
Подсказка

Используйте цикл while для повторения операций, словарь для хранения доступных операций, условные операторы для проверки ввода. Не забудьте обработать случаи деления на ноль и некорректного ввода.

Решение 
def main():
    print("=" * 50)
    print("ИНТЕРАКТИВНЫЙ КАЛЬКУЛЯТОР")
    print("=" * 50)
    print("Добро пожаловать в интерактивный калькулятор!")
    
    # Определяем доступные операции
    operations = {
        '1': ('Сложение', lambda x, y: x + y),
        '2': ('Вычитание', lambda x, y: x - y),
        '3': ('Умножение', lambda x, y: x * y),
        '4': ('Деление', lambda x, y: x / y if y != 0 else "Ошибка: деление на ноль"),
        '5': ('Возведение в степень', lambda x, y: x ** y),
        '6': ('Остаток от деления', lambda x, y: x % y if y != 0 else "Ошибка: деление на ноль"),
        '7': ('Целочисленное деление', lambda x, y: x // y if y != 0 else "Ошибка: деление на ноль")
    }
    
    continue_calc = True
    
    while continue_calc:
        # Показываем меню
        print("\nДоступные операции:")
        for key, (name, _) in operations.items():
            print(f"{key}. {name}")
        print("0. Выход")
        
        # Получаем выбор пользователя
        choice = input("\nВыберите операцию (0-7): ")
        
        # Проверяем выбор
        if choice == '0':
            print("Спасибо за использование калькулятора! До свидания!")
            break
        
        if choice not in operations:
            print("Ошибка: Неверный выбор операции. Пожалуйста, выберите от 0 до 7.")
            continue
        
        # Получаем название и функцию выбранной операции
        operation_name, operation_func = operations[choice]
        
        # Запрашиваем числа
        try:
            num1 = float(input("Введите первое число: "))
            num2 = float(input("Введите второе число: "))
            
            # Выполняем операцию
            result = operation_func(num1, num2)
            
            # Проверяем результат на ошибку
            if isinstance(result, str):
                print(result)
            else:
                # Форматируем вывод в зависимости от результата
                if result.is_integer():
                    result = int(result)
                
                print(f"\nРезультат операции '{operation_name}':")
                print(f"{num1} {get_operation_symbol(choice)} {num2} = {result}")
                
                # Дополнительная информация о результате
                if result > 0:
                    print("Результат положительный")
                elif result < 0:
                    print("Результат отрицательный")
                else:
                    print("Результат равен нулю")
                
                if isinstance(result, int) and result != 0:
                    if result % 2 == 0:
                        print("Результат - четное число")
                    else:
                        print("Результат - нечетное число")
        
        except ValueError:
            print("Ошибка: Введены некорректные числа. Пожалуйста, попробуйте снова.")
            continue
        
        # Спрашиваем, хочет ли пользователь продолжить
        continue_choice = input("\nХотите выполнить еще одну операцию? (да/нет): ").lower()
        continue_calc = continue_choice == 'да' or continue_choice == 'y' or continue_choice == 'yes'

def get_operation_symbol(choice):
    symbols = {
        '1': '+',
        '2': '-',
        '3': '*',
        '4': '/',
        '5': '**',
        '6': '%',
        '7': '//'
    }
    return symbols.get(choice, '?')

if __name__ == "__main__":
    main()
